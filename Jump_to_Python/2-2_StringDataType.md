# 2-2. 문자열 자료형
* 이스케이프 코드

|코드|설명|
|---|---|
|\n|줄바꿈|
|\t|탭 간격|
|\\\\|문자열에 \포함|
|"\\\'"|문자열에 '포함|
|\\\"|문자열에 "포함|
|\r|캐리지 리턴(커서를 현재줄 가장 앞으로|
|\f|폼 피드(커서 위치 그대로 두고 줄 바꿈)|
|\v|수직탭|
|\a|벨소리|
|\b|백스페이스|
|\000|널문자|

* raw string
```python
print(r'\\hello\nworld\\')
\\hello\nworld\\
# 이스케이프 기능을 무시하고 그대로 문자열을 추가할 때, 문자열 앞에 r 추가
```
